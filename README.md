Medicine Inventory Management System (Java-GUI+OOP)
A complete desktop application built using Java Swing, designed to manage medicines efficiently through an intuitive graphical interface.
This project demonstrates clean Object-Oriented Programming (OOP), a modular architecture, file-based persistence, and user authentication.
